# ⚖️📊 Dashboard de Performance Jurídica

Projeto exclusivo e totalmente customizado para profissionais da área do Direito, desenvolvido em **Power BI** com dados fictícios gerados por inteligência artificial e organizados em planilhas do **Excel**.

---

## 📁 Sobre o Projeto

Este dashboard foi pensado para entregar uma visão **clara, objetiva e estratégica** da atuação jurídica, com indicadores essenciais para o dia a dia dos escritórios de advocacia.

---

## 🔍 Recursos Aplicados

| Recurso | Descrição |
|---|---|
| 📌 **KPIs Principais** | Indicadores de clientes ativos, total de advogados, processos em andamento e concluídos |
| 🔎 **Filtros Dinâmicos** | Filtros por ano ou por advogado, facilitando o acompanhamento por período ou responsável |
| 🔤 **TextFilter** | Busca por nome do cliente, tipo de processo ou status |
| 💰 **Consulta de Honorários** | Consulta por número da OAB, garantindo controle financeiro personalizado |
| 📈 **Gráfico de Previsão** | Gráfico de linha com previsão de custos e honorários a serem recebidos |

---

## 📈 Objetivo

Um painel completo para:

- ✅ Monitorar resultados
- ✅ Analisar performance da equipe jurídica
- ✅ Tomar decisões com mais confiança
- ✅ Planejar financeiramente com base em projeções de custos e honorários

Tudo com uma visualização **prática e intuitiva**.

---

## 🛠️ Tecnologias Utilizadas

- **Power BI** — Desenvolvimento do dashboard e visualizações
- **Microsoft Excel** — Organização e estruturação dos dados
- **Inteligência Artificial** — Geração dos dados fictícios utilizados no projeto

---

## 💼 Para Quem é Este Projeto?

Este dashboard é ideal para:

- Advogados que desejam ter controle total da sua atuação
- Gestores de escritórios jurídicos que precisam de uma visão estratégica
- Profissionais do Direito que buscam centralizar informações em um único painel

---

## 📌 Observação

> Os dados utilizados neste projeto são **fictícios**, gerados por inteligência artificial apenas para fins de demonstração. Nenhuma informação real de clientes ou processos foi utilizada.
